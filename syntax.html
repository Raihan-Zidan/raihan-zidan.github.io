Search Syntax
Find price
Put $ in front of a number. For example: shoe $100.
Search specific site
Put " site:" in front of the site or domain. For example, site:youtube.com or site:.com.
Searching for files with a specific format
Put it after search query. for Example, filetype:pdf. Supported file types: pdf, doc(x), xls(x), ppt(x), html.
Search for sites with titles containing specific words
For example. intitle:example, The page title includes the word "example".
Search for sites with urls containing specific words
For example. inurl:example, The page url includes the word "example".
What is the above syntax function?
Searching by including the above syntax will help search engines to show more detail about what you are looking for.
Does the syntax have to be included?
Do not have to. It depends on what you want to know.
